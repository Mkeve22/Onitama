Onitama

Java Swing alapú Onitama játék, az eredeti japán stratégiai táblajáték digitális változata.
Projectmunka aminek a céljai a következők:

## ⚙️ Fő funkciók
- 5x5-ös játéktábla kirajzolása
- Kártyák és lépések kezelése
- Mentés és betöltés XML vagy JSON fájlból
- Menü kezelése (új játék, kilépés stb.)
- Játékosok és figurák megjelenítése
